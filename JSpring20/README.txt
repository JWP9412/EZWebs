[AOP]
1. Aspect Oriented Programming
2. 관점 지향 프로그래밍
3. 주기능과 보조 기능을 분리 

(용어)
1. aspect : 구현하고자 하는 보조 기능
2. advice : aspect의 실제 구현체를 의미, 메서드 호출을 기준으로 여러 지점에서 실행
3. jointpoint : advice를 적용하는 지점
4. pointcut : advice가 적용되는 대상 지정
5. target : advice가 적용되는 클래스
6. weaving : advice를 주기능에 적용하는 것
