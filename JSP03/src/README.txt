[액션 태그]

1. Include 액션 태그
	- JSP를 포함
	- <jsp:include>

2. 포워드 액션 태그
	- RequestDispatcher 포워딩
	- <jsp.forward>
	
3. 유즈빈 액션 태그
	- 자바의 new 연산자를 대신하는 태그
	- <jsp:useBean>
	
4. Set 프로퍼티 액션 태그
	- 자바의 setter 
	- <jsp:setProperty>
	
5. Get 프로퍼티 액션 태그
	- 자바의 getter
	- <jsp.getProperty>
